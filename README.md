# WANDER

A Text Adventure Service

## Usage

To really use this you need a game, and there is currently not one that I hace permission to share. I am working on that.

To build you'd use the following:

```bash
$ yarn install
$ yarn main start

```

## Development

```bash
$ yarn watch
```
